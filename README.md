# The Hippocratic Code of Conduct

A politically neutral open-source Code of Conduct rooted in ancient philosophy. Built on the universal principle of "First, do no harm" to foster a respectful, drama-free developer community.

## Why does this exist?

Modern open-source communities sometimes get bogged down in contemporary political or cultural debates. The Hippocratic Code of Conduct aims to bypass this friction by anchoring community guidelines in universal, time-tested human virtues.

It draws inspiration from:

- **The Medical Tradition:** Primum non nocere ("First, do no harm").

- **Eastern Philosophy:** Ahimsa (non-injury), Right Speech, and Ren (benevolence).

- **Stoicism:** Sympatheia (mutual interdependence) and the Dichotomy of Control.

The goal is simple: put forward your best behavior, focus on the code, and don't be an asshole.

## How to adopt this for your project

Adopting this Code of Conduct is simple and requires no attribution:

1. Download or copy the contents of the `CODE_OF_CONDUCT.md` file.

2. Place the file in the root directory (or `.github/` folder) of your repository.

3. Open the file and scroll to the bottom.

4. Replace the `[Maintainers: Please insert instructions here...]` placeholder in the Reporting Issues section with your project's actual contact email or reporting instructions.

5. Commit and push! GitHub will automatically detect it and display it to contributors.
